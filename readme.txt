Plugin completo para el proceso de adhesión de clientes con calculadora de presupuestos, pagos y firma digital de contratos.

== Description ==

Este plugin facilita el proceso completo de adhesión de clientes, incluyendo:

* Calculadora de presupuestos personalizable
* Sistema de registro y login de usuarios
* Integración con Redsys para pagos
* Firma digital de contratos con DocuSign
* Panel de administración completo
* Gestión de documentos editable

== Installation ==

1. Sube el plugin a la carpeta `/wp-content/plugins/`
2. Activa el plugin desde el menú 'Plugins' de WordPress
3. Configura las integraciones desde Adhesión > Ajustes
